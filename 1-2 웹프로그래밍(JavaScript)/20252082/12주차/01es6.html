템플릿 문자열

` ${변수}` 

        const name="leesunshin";
        const age=53;
        document.write("name : "+name+", age : "+age);
        
        // ES6문법 템플릿 문자열
        document.write(`name : ${name},  age : ${age}`);    //답
        //document.write(`name : ${name} "\n"  age : ${age}`); -> \n은 사용 안됨 br만 가능!!
